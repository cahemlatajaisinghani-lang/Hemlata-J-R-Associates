# Push to GitHub - Step by Step Instructions

## You're Almost There! Follow These Simple Steps:

### STEP 1: Open Git Pane in Replit
1. Look at the left side of Replit
2. Find and click on the **Git** icon (it looks like a branch/network symbol)
3. A panel will open on the left showing Git options

### STEP 2: Stage All Changes
1. In the Git panel, you'll see "Unstaged changes"
2. Look for a **+** button or **Stage all** button
3. Click it
4. All files will move to "Staged changes"

### STEP 3: Write a Commit Message
1. You'll see a text box that says "Commit message" or similar
2. Type this exactly:
```
Complete professional website for Hemlata J R & Associates
```

### STEP 4: Click Commit
1. Look for a blue **Commit** button
2. Click it
3. Wait a few seconds

### STEP 5: Push to GitHub
1. After committing, look for a **Push** button (might say "Push to origin" or just "Push")
2. Click it
3. Wait 30 seconds
