# Simple - How to Find Git in Replit

## Follow EXACTLY:

### STEP 1: Look at Top Right of Replit
1. Find the three dots menu **â‹¯** (it's at the top right)
2. Click on it

### STEP 2: Look for Tools
1. A dropdown menu will appear
2. Look for **Tools** option
3. Click on it

### STEP 3: Select Git
1. You'll see a list of tools
2. Click on **Git**
3. The Git pane will open on the left side

### STEP 4: You Should Now See:
- Changes section with your files
- A text box to write a message
- A **Commit** button
- A **Push** button

---

## If that doesn't work, try this:

### Alternative: Look in Left Sidebar
1. Look at the very left side of Replit (the sidebar with icons)
2. Scroll down to the bottom of those icons
3. Look for any icon that looks like:
   - Branches/network
   - Or a square/circle
4. Click each one until you find "Git"

---

## Still can't find it? Try this:

1. Click on your **project name** at top left
2. Look for "Versions" or "Version Control"
3. Click it

---

## Once Git Opens:

1. You'll see your files listed under "Changes"
2. Click on each file (or look for a checkbox next to all files)
3. All files will be selected
4. Write a message:
```
My professional website
```
5. Click **Commit**
6. Click **Push**
